Physics Notes

The 2/pi compression decays intensity each step. The critical alpha is where neighbor coupling exactly compensates decay.

Above alpha_crit the wavefront advances at 1 node/step — the system's speed of light.

Map to real c: set dx/dt = 3e8 m/s. At 1m spacing, each step = 3.33 ns.
